# Mallorca

A static travel guide landing page.

## Serve locally

1. Open a terminal in `mallorca/`.
2. Install dependencies:

```bash
npm install
```

3. Start the local server:

```bash
npm run start
```

4. Open `http://localhost:8080` in your browser.

## Alternative without npm

If you do not want to use Node, you can also serve the folder with Python:

```bash
cd mallorca
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
